# Sales & Customer Performance Analysis | Power BI

## Business Context
The objective of this project was to analyze sales performance, customer behavior, and profitability to support data-driven business decisions using a retail dataset.

## Tools
- Power BI
- DAX
- Excel

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- Total Orders
- Average Order Value (AOV)
- Year-to-Date (YTD) Sales

## Key Insights
- Furniture is the highest-selling category but generates the lowest profit margin.
- Bookcases sub-category consistently shows negative profitability.
- Sales are regionally concentrated, with Texas representing the highest customer spend.
- Profitability varies significantly across categories, highlighting optimization opportunities.

## Outcome
Delivered an end-to-end Power BI dashboard that enables stakeholders to identify low-margin products, regional sales patterns, and profitability drivers.
